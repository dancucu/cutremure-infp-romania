## SIMPLIFIED CONFIG FLOW - Pattern Modern

Am simplificat complet structura config flow folosind pattern-ul oficial modern Home Assistant.

### Schimbări Critice:
1. **Domain ca parametru în clasă**: `class CutremureConfigFlow(config_entries.ConfigFlow, domain="cutremure_infp")`
2. **Eliminat decorator învechit**: Fără `@HANDLERS.register()`
3. **Eliminat import explicit**: Fără import în `__init__.py` - Home Assistant descoperă automat
4. **String literal peste tot**: Fără variabile DOMAIN, doar string-uri directe

### Cod:
```python
class CutremureConfigFlow(config_entries.ConfigFlow, domain="cutremure_infp"):
    VERSION = 1
    
    async def async_step_user(self, user_input=None):
        if user_input is not None:
            await self.async_set_unique_id("cutremure_infp")
            ...
```

Acesta e pattern-ul oficial din documentația Home Assistant 2024+. Ar trebui să funcționeze.
