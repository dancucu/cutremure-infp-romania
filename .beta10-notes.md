## Fix Eroare Sintaxă din Beta.9

Corectează eroarea de sintaxă din versiunea precedentă:
- Restabilit corect: `await self.async_set_unique_id("cutremure_infp")`
- Restabilit corect: `self._abort_if_unique_id_configured()`

Pattern-ul rămâne același ca în beta.9:
```python
class CutremureConfigFlow(config_entries.ConfigFlow, domain="cutremure_infp"):
    VERSION = 1
    
    async def async_step_user(self, user_input=None):
        if user_input is not None:
            await self.async_set_unique_id("cutremure_infp")
            self._abort_if_unique_id_configured()
            return self.async_create_entry(title="Cutremure INFP", data=user_input)
```

**Folosește această versiune, nu beta.9!**
